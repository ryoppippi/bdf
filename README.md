# bdf.fish
A fish plugin to cd to the upper directories faster using fzf

Inspired by [fish-bd](https://github.com/0rax/fish-bd) and [enhancd](https://github.com/babarot/enhancd)

```fish
fisher install ryoppippi/bdf.fish
```
